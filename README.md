# Marconde Finanças

> Plataforma web de gestão financeira — Gerada pelo AegisHub Builder

## Stack
- Frontend: React + TypeScript + Tailwind CSS
- Backend: Supabase Edge Functions
- Database: Supabase PostgreSQL

## Módulos
1. **Dashboard** — Painel principal com cards e gráficos
2. **Transactions** — Cadastro de receitas e despesas
3. **Categories** — Gestão de categorias financeiras
4. **Insights** — Inteligência financeira e alertas
5. **Reports** — Histórico e relatórios por período

## Rodar Localmente
```bash
npm install
npm run dev
```
